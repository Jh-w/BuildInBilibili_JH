# B-EP0 预告视频脚本：「我要做一个会骂人的高考志愿AI」

> 发布日期：2026-06-09（高考结束当天）
> 视频类型：概念预告 / 开发预告
> 目标时长：2分30秒 - 3分钟
> 核心钩子：「别的AI帮你算分，我让AI帮你认清自己」


## 拍摄/制作备忘

### 画面素材需求

| 时间点 | 素材 | 状态 |
|--------|------|------|
| 0:00 | 黑底白字打字机效果 | 剪辑时制作 |
| 0:08 | 搜索引擎截图滚动 | 录屏 |
| 0:20 | 大厂AI产品界面截图 | 录屏/截图 |
| 1:35 | 产品模拟界面 | **需提前做 HTML 原型（静态即可）** |
| 2:25 | C 跑步人格测试界面 | 截图/录屏 |
| 2:38 | 片尾黑屏大字 | 剪辑时制作 |

### 需提前准备

1. **产品模拟器（静态 HTML）** — 不是真实产品，只是录屏用的「假界面」，有输入框+聊天气泡动画。做一个简单的 HTML 页面，录成 GIF 或直接录屏。
2. **大厂产品截图** — 腾讯元宝高考通、夸克高考、百度文心、字节豆包，各截一张首页。
3. **C 跑步人格 2.0 预览图** — 一张对话界面的概念图（可以只是设计稿）。

### 录音建议

- 语气：轻松 + 贱 + 真诚。不是播音腔，是跟朋友吐槽的语气。
- 「625分连门卫的微信都加不上」这句要说得特别自然，像随口一说。
- BGM 分层：开场→低沉悬念；中段→轻快搞怪；结尾→热血期待。

### 剪辑要点

- 节奏快，不废话。3分钟是上限，2分半理想。
- 打字机效果 + 音效是记忆点，不要省略。
- 毒舌对话那段的三个气泡依次弹出 + 打字机音效 = 核心钩子。
