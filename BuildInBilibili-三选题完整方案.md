# 🅰️ AI 身体翻译官

## 一句话定位

**用 BME 专业知识把身体信号翻译成人话——让你的身体自己开口吐槽你。**

---

## 产品方案

### 核心功能

| 功能 | 描述 | 开发量 |
|------|------|--------|
| **身体信号输入** | 用户输入「症状/感受/数据」（如「昨晚睡了5小时，今天HRV降了15，左膝有点酸」） | ⭐ 低 |
| **AI 翻译引擎** | 多角色翻译：毒舌版 / 科普版 / 教练版 / 妈妈版。BME 专业知识注入 system prompt | ⭐⭐ 中 |
| **身体日报** | 每天汇总运动/睡眠/饮食数据，生成当日身体吐槽报告 | ⭐⭐⭐ 高（需要数据接入，可二期） |
| **器官拟人化画廊** | 每个器官有固定人设（心脏=卷王 996永不罢工、膝盖=财务总监精打细算、胃=美食评审员），用户数据激活不同器官发言 | ⭐⭐ 中 |
| **分享卡片** | 复用跑步人格测试的 Canvas 方案，生成可分享的「今日身体发言」卡片 | ⭐ 低 |

### MVP 范围（2 周可上线）

```
第一周：器官人设库（6-8个核心器官）+ AI 翻译 prompt 调优 + 输入界面
第二周：分享卡片 + 多角色切换 + 部署上线
```

### 器官人设库（核心卖点）

| 器官 | 人设 | 典型发言 |
|------|------|---------|
| ❤️ 心脏 | 卷王，996 永不罢工，但会偷偷记仇 | 「你昨晚熬夜到三点，我今天多跳了 3000 下你知道吗」 |
| 🦵 膝盖 | 财务总监，精打细算每一分软骨 | 「你今天跑那 15 公里，按折旧率算我少了 0.003mm 软骨，你赔」 |
| 🧠 大脑 | 被迫营业的 CEO | 「你说你焦虑，我 cortisol 都飙到 18 了，你问我怎么办？」 |
| 🍖 肌肉 | 老实打工人，不抱怨但会酸痛 | 「你昨天深蹲 80kg x 5x5，我现在还在修复，明天别安排了」 |
| 🛌 睡眠系统 | 傲娇管理员 | 「昨晚 REM 只有 1.2h，你以为 5 小时就够？想多了」 |
| 🩸 免疫系统 | 社恐保安，不想加班 | 「你又喝酒了是吧，我白细胞现在全员出动，明天开会请假」 |

### BME 知识注入方式

每个器官发言背后有真实的生理学依据，用 BME 知识确保：
- HRV 下降 → 自主神经系统的真实反馈
- 膝盖酸 → 软骨/半月板的力学负荷边界
- 睡眠不足 → REM/NREM 对恢复的具体影响

**这是别人做不了的核心壁垒。**

### 商业模式（长线）

- 免费版：每日 1 次身体翻译 + 2 个器官角色
- Pro 版（会员）：无限制 + 全器官 + 身体趋势报告 + 数据接入（Garmin/Apple Health）

# 🅱️ AI 高考志愿毒舌助手

## 一句话定位

**一个敢说真话的 AI 高考志愿顾问——不画饼、不鸡汤、用数据打脸。**

---

## 产品方案

### 核心功能

| 功能 | 描述 | 开发量 |
|------|------|--------|
| **专业真相卡** | 输入任意专业，AI 输出这个专业的「残酷真相」：真实就业率、平均薪资、5 年后转行率、学长学姐原话 | ⭐⭐ 中 |
| **毒舌匹配** | 输入分数+兴趣+性格，AI 给出 3 个「你应该报的」+ 3 个「你千万别报的」 | ⭐⭐ 中 |
| **专业对比器** | 两个专业并排 PK：钱、累不累、能不能考公、35 岁门槛 | ⭐ 低 |
| **避坑指南** | 预设 Top 20 「高开低走」专业排行榜，每个专业配一句毒舌评论 | ⭐ 低 |
| **分享卡片** | 「我被 AI 推荐去学____」结果卡片 | ⭐ 低 |

### MVP 范围（10 天可上线）

```
第 1-2 天：数据采集（各专业就业率/薪资/转行率等公开数据）
第 3-4 天：AI prompt 调教（毒舌人格 + 数据注入）
第 5-7 天：前端开发（输入界面、结果展示、分享卡片）
第 8-9 天：部署 + 内测 + 调整
第 10 天：上线
```

### 核心竞争力

1. **时效性**：6 月底出分，现在就是最佳上线窗口
2. **争议性**：毒舌 = 天然传播引擎。「你千万别学这个专业」是每个高三生都想点开的标题
3. **数据背书**：不是瞎吐槽，每个「毒舌」背后有就业数据支撑

### 风险点

- 被某些学校/专业投诉（用数据说话可以降低风险）
- 用户对「AI 推荐的专业」期望过高 → 每个结果加免责声明
- 高考后流量窗口短 → 长期价值有限，是纯流量品

### 传播策略

- 做「专业真相」系列短视频（从产品里截取部分结果）
- 标题模板：「AI 说千万别学____专业的 3 个理由」
- 评论区引导用户输入自己的情况，AI 给出结果

# 🅲️ 跑步人格测试 2.0（AI 对话版）

## 一句话定位

**跑步人格测试的升级版——测完人格之后，跟 AI 聊聊你的跑步人生。**

---

## 产品方案

### 一期 vs 二期

| 维度 | 1.0（已上线） | 2.0（本期加） |
|------|-------------|-------------|
| 测试流程 | ✅ 8 题 → 16 型人格 | 保持不变 |
| 结果页 | ✅ 人格标签 + 训练建议 | **+ AI 深度对话入口** |
| 分享 | ✅ Canvas 卡片 | 保持不变 |
| **AI 对话** | ❌ 无 | **测完后，AI 以人格角色的口吻跟你聊跑步** |
| **数据看板** | ❌ 无 | **新增个人跑步人格分析报告** |

### 2.0 新增功能

| 功能 | 描述 | 开发量 |
|------|------|--------|
| **AI 人格角色对话** | 测完后，AI 以你的人格角色（如「赛道卷王🏆」）的口吻跟你聊：为什么你是这个类型、你的训练建议、你的跑步故事 | ⭐⭐ 中 |
| **AI 教练对话** | 更深一层的聊天：输入你最近的训练感受，AI 教练（BME 知识注入）给出个性化建议 | ⭐⭐ 中 |
| **人格分析报告** | 基于四维度得分，生成 500 字的人格深度报告（AI 生成） | ⭐ 低 |

### MVP 范围（1 周可上线）

```
第 1 天：在现有 H5 结果页加 AI 对话入口
第 2-3 天：AI 人格角色对话 prompt（16 个角色各设计对话风格）
第 4-5 天：AI 教练对话 prompt（BME 知识注入）
第 6 天：人格分析报告生成
第 7 天：部署上线
```

### 核心优势

1. **已有产品基础**：不是从零开始，是在已有 H5 上加功能，开发量最小
2. **已有种子用户**：如果 1.0 已经有一些用户，2.0 可以召回
3. **天然 Build in Public 故事**：「我上个月做了一个东西，现在我要让它变聪明」

### 劣势

- 新鲜感不如 A 和 B，是「升级」而非「新品」
- 受众仍然局限在跑步人群
- 跟你求职方向（AI 健康数据产品）关联度最高

# 三维度对比决策表

| 维度 | 🅰️ AI 身体翻译官 | 🅱️ AI 高考志愿毒舌助手 | 🅲️ 跑步人格测试 2.0 |
|------|------------------|----------------------|---------------------|
| **开发周期** | 2 周 | 10 天 | 1 周 |
| **技术壁垒** | ⭐⭐⭐⭐⭐ BME知识只有你有 | ⭐⭐ 数据采集 + prompt | ⭐⭐⭐ 已有基础 |
| **长期价值** | ⭐⭐⭐⭐⭐ 可迭代为独立产品 | ⭐⭐ 季节性，6月底后流量断崖 | ⭐⭐⭐⭐ H5持续运营 |
| **传播潜力** | ⭐⭐⭐⭐ 器官拟人可出圈 | ⭐⭐⭐⭐⭐ 高考争议天然传播 | ⭐⭐⭐ 垂直但精准 |
| **求职关联度** | ⭐⭐⭐⭐⭐ 直接对口运动健康数据PM | ⭐⭐ 纯流量品 | ⭐⭐⭐⭐⭐ 数据产品迭代能力 |

| **B站内容可持续性** | ⭐⭐⭐⭐⭐ 每天一个器官都能出视频 | ⭐⭐ 高考后没话题了 | ⭐⭐⭐⭐ 跑步社区长期内容 |
| **单枪匹马能做吗** | ✅ 前端+AI prompt | ✅ 前端+AI prompt | ✅ 已有代码，加量即可 |

## 我的综合建议

如果只能选一个：**A（AI身体翻译官）**。理由三条：

1. **你是唯一能做的人**。市面上做高考 AI 的人很多，做跑步测试的人也可能有。但「BME科班 + AI产品 + 运动达人 + 能独立开发」做身体翻译，你就是这个交叉点上唯一的人。
2. **跟你的求职目标完美咬合**。这个产品投 Keep/华为/小米 都是直接弹药——「你看，我连器官拟人化产品都做出来了，我对健康数据的产品化理解不是纸上谈兵」。
3. **B 站内容能持续出一年**。高考志愿助手 6 月后就没人看了，身体翻译官可以每周出一集「某个器官的内心独白」。

如果追求最快出成果：**C（跑步人格2.0）**——一周上线，复用已有代码，风险最低。

如果追求流量爆发：**B（高考志愿毒舌）**——窗口就在眼前，但 7 月后就没了。
