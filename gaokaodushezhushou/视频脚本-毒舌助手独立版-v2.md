# 视频脚本：毒舌助手独立版 v2 · 验证娱乐向

> **v1 → v2 策略变更**：从「帮你做选择」→「帮你验证选择」。核心假设：志愿填报窗口已近尾声，主流用户是「已填完的人」而非「还在纠结的人」。策略转向娱乐共鸣+验证心态，去时效化。
>
> 平台：B站（首发） + 小红书（同步，剪60秒竖版精简版）  
> 时长：B站 3-4 分钟 / 小红书 ≤60秒  
> 格式：B站横屏 16:9 / 小红书竖屏 9:16  
> 话题：无品牌限制，建议 #高考志愿 #AI锐评 #专业真相  
> 发布日期：7月2日-7月5日（高考季尾部，志愿提交后的话题余温）  
> 产品：高考毒舌助手（自有产品）

| 时效性 | 绑死填报窗口 | 去时效化，吃7月整个高考话题尾部 |

### 00:20-01:20 · 主体1：三段式毒舌锐评（核心高光）

**用3-4个热门专业做案例，展示完整三段式：**

| 画面 | 字幕/旁白 |
|------|----------|
| **案例1：工商管理** | 「第一个——工商管理」 |
| AI毒舌回复（红框） | 「"恭喜你选了全中国最不缺人的专业，毕业起薪和便利店店长差不多"」 |
| AI数据回复（蓝框） | 「然后给数据：2025届平均起薪4,200，对口就业率38%」 |
| AI自首（绿框） | 「最后自己认——"其实你让我推荐我也推荐不出花来，这专业就是什么都会什么都不精"」 |
| 画外音 | **「报了工商管理的同学，对不住了。」** |
| **案例2：生物医学工程** | 「第二个——生物医学工程，我的本专业」 |
| AI毒舌 | 「"名字里有'医学'又有'工程'，毕业了医院不认你，药企不认你，IT嫌你半路出家"」 |
| AI自首 | 「"不过如果你真的热爱，读研读博走医疗器械研发，还是能打的"」 |
| 画外音 | **「这句话，我当年没人告诉我。」** |
| **案例3：电子商务** | 「还有一个——电子商务」 |
| AI毒舌 | 「"这专业最大的价值，就是让你发现你不需要这个专业也能做电商"」 |
| **案例4（可选）：英语/小语种** | 「英语专业，AI锐评——」 |
| AI毒舌 | 「"DeepL已经比大部分英专生翻译得好了，但你如果辅修一个法律或者金融，立刻变成稀缺物种"」 |
| AI自首 | 「"不过纯英语专业确实难，除非你走同传或者高校教职，那另说"」 |

> 🎬 三段式颜色体系：😈 毒舌=珊瑚红边框、📊 数据=冷蓝色边框、🤖 自首=绿色边框
> 🎬 关键变化：每段结尾加一句画外音「评论」，增强娱乐感和人格
