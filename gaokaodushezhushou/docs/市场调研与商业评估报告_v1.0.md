## 1. 市场全景

### 1.1 核心数据

| 指标 | 数据 | 来源 |
|------|------|------|
| 2026年高考报名人数 | **1290万人** | 多渠道交叉验证 |
| 面向高校数量 | ~3000所 | 教育部公开数据 |
| 面向专业数量 | 1600+个 | 教育部学科目录 |
| 大厂入场数 | ≥4家（腾讯/阿里/百度/字节） | 产品体验+媒体报道 |
| 高考志愿付费市场规模 | 估算100亿+（含一对一规划师） | 行业报道 |

### 1.2 时间窗口

```
2026 高考时间线
─────────────────────────────────────────────────────────────
6/7-9   高考考试          大厂AI工具主动锁死部分功能（防作弊）
6/10-22 等待出分           焦虑期：搜索量激增，内容消费旺盛
6/23    出分日 🔥          全年流量峰值日之一
6/23-7/5 志愿填报窗口     搜索+决策高峰期（仅13天！）
7/5+    志愿填报结束        流量断崖下跌
─────────────────────────────────────────────────────────────
```

**关键洞察**：高考志愿赛道的有效窗口仅 **13天**（6/23-7/5），但流量密度极高。产品形态必须适配「短窗口高爆发」特征。

### 1.3 市场结构

| 层级 | 代表玩家 | 价格 | 用户规模 |
|------|---------|------|----------|
| 🔴 高端一对一 | 志愿规划师 | 5000-20000元/人 | 极小（经济门槛） |
| 🟡 大厂免费工具 | 夸克/元宝/百度/豆包 | 免费 | 极大（千万级） |
| 🟢 内容/IP型 | **（空白）** | 免费 | 潜力巨大 |

## 2. 竞品深度分析

### 2.1 已入场玩家

| 玩家 | 产品名 | 核心功能 | 定位 |
|------|--------|---------|------|
| **腾讯** | 元宝高考通（×QQ浏览器） | AI咨询师Agent + 志愿匹配 | 全能AI顾问 |
| **阿里/夸克** | 夸克高考频道 | 搜索、智能选志愿、志愿表生成 | 免费工具矩阵 |
| **百度** | 文心助手·高考模块 | 搜索入口+AI推荐 | 搜索流量分发 |
| **字节/豆包** | 豆包 | 志愿填报功能 | 通用AI延伸 |

### 2.2 竞品共同特征（= 可打突破口）

| 维度 | 所有竞品 | 我们的机会 |
|------|---------|-----------|
| **产品形态** | 数据查询器 / 匹配计算器 / 报告生成器 | **人格化聊天助手** |
| **语气** | 中性、客观、"官方感" | **毒舌自嘲、像朋友** |
| **用户情感** | 解决焦虑的方式=给更多数据 | **用幽默消解焦虑** |
| **传播性** | 几乎为零（没人会截图高考工具发朋友圈） | **天然截图传播基因** |
| **人格** | 无性格、无态度 | **鲜明人设（毒舌但善良）** |
| **娱乐性** | 零 | **段子式输出+自嘲** |

### 2.3 十篇文章中从未出现的关键词

> 幽默 / 搞笑 / 有梗 / 人格化 / 人设 / 性格 / 吐槽 / 毒舌 / 娱乐 / 好玩 / 有趣 / 传播 / 出圈 / 共情

**→ 这意味着「AI高考志愿 × 人格化娱乐内容」是一个完全蓝海。**

## 3. 用户需求洞察

### 3.1 用户分层

| 画像 | 占比估算 | 核心需求 | 情感状态 |
|------|----------|---------|----------|
| 🟢 **高分考生**（≥本科线+30分） | ~30% | 冲更好学校，需要精准对标 | 有底气，愿意被调侃 |
| 🟡 **中分段考生**（本科线±30分） | ~40% | 焦虑选校，怕滑档 | 焦虑但还有选择 |
| 🔴 **低分段考生**（<本科线-30分） | ~30% | 不知道还有什么路 | 脆弱，需要鼓励和实际路径 |

### 3.2 核心痛点

| 类型 | 痛点 | 当前市场方案 | 我们的解法 |
|------|------|------------|-----------|
| 🔴 信息焦虑 | 分数出来了，3000所学校1600个专业不知道怎么选 | 给你一个搜索框+匹配算法 | 用人话告诉你真相+让你笑出来 |
| 🔴 认知幻觉 | 考生对自己分数的竞争力缺乏现实认知 | 冷冰冰的冲稳保列表 | 毒舌暴击+数据佐证→认清现实 |
| 🟡 专业迷茫 | 专业名字看不懂，不知道学了能干嘛 | 官方介绍 | 专业翻译官：真实生活毒舌版 |
| 🟡 决策孤独 | 父母不懂、二舅瞎建议 | 无 | AI自嘲式陪伴：「比二舅靠谱」 |
| 🟢 表达欲 | 想吐槽/晒自己的分数处境 | 无 | 截图分享卡片：「AI说我XXX」 |

## 4. 差异化机会验证

### 4.1 换赛道策略

```
大厂在打「工具战」                    我们在打「内容战」
─────────────────                    ─────────────────
竞争维度：数据准不准                  竞争维度：内容传不传
          匹配准不准                            人设记不记住
          功能多不多                            看完笑没笑
          ↓                                    ↓
用户不会转发工具                   用户会截图发朋友圈/B站动态
```

### 4.2 B站平台匹配度

| 平台特征 | 本项目匹配度 |
|---------|------------|
| 年轻用户为主（18-24岁 = 高考目标人群） | ✅ 完美匹配 |
| 对"毒舌""吐槽"内容接受度极高 | ✅ 天然适配 |
| 社区有转发/截图传播习惯 | ✅ 传播基因 |
| B站AI创造公开赛正在进行 | ✅ 比赛窗口 |
| 高考话题是B站站内热点 | ✅ 流量窗口 |

### 4.3 市场规模估算

| 层级 | 预估触达 | 转化率 | 预估用户 |
|------|---------|--------|----------|
| 高考考生总量 | 1290万 | — | — |
| B站用户中考生/家长 | ~500万（估算） | — | — |
| 视频播放触达 | ~100万 | 3-5%点击率 | 3-5万 |
| 产品使用 | 3-5万 | 100% | 3-5万UV |

**结论**：作为引流工具，3-5万目标UV足以对主合集产生显著流量贡献。

## 5. 市场风险与合规

### 5.1 风险评估矩阵

| 风险 | 概率 | 影响 | 评级 | 描述 |
|------|------|------|------|------|
| 舆情反噬（毒舌被截图断章取义） | 中 | 🔴 高 | ⚠️ 8/10 | 有人截取毒舌部分发微博「AI嘲讽考生」 |
| 考生情感伤害 | 中 | 🟡 中 | ⚠️ 6/10 | 低分考生被怼后心态崩溃 |
| 家长举报 | 低 | 🟡 中 | ⚠️ 4/10 | 「你们这个AI怎么能这么跟孩子说话」 |
| B站社区举报 | 低 | 🟡 中 | ⚠️ 4/10 | 「恶意博眼球」 |
| 比赛取消资格 | 低 | 🔴 高 | ⚠️ 5/10 | 规则禁止「恶意博眼球」 |
| 时间窗口过短 | 确定 | 🟡 中 | ⚠️ 6/10 | 有效窗口仅13天 |

### 5.2 合规要求

| 合规维度 | 本项目状态 | 要求 |
|---------|-----------|------|
| 个人信息保护（PIPL） | ⚠️ 需注意 | 不收集用户真实姓名/准考证号，仅收集分数+省份 |
| B站平台政策 | ⚠️ 需注意 | 视频标题封面不能有「恶意博眼球」表述 |
| 比赛规则 | ⚠️ 需注意 | 产品可独立运营，不强制挂活动标签 |
| 未成年人保护 | ✅ 已覆盖 | System prompt 含低分段保护+内容红线 |

## 6. 商业模式分析

### 6.1 当前阶段（2026年6月）

| 角色 | 商业价值 |
|------|---------|
| **引流工具** | 将高考流量导入B站主合集，提升参赛投币数 |
| **内容IP** | 建立"毒舌AI"人设，为后续内容打下品牌基础 |
| **社交货币** | 截图分享带来二次传播，降低获客成本 |

**结论**：MVP阶段**不追求直接变现**，价值体现为「主合集的流量发动机」。

### 6.2 未来变现可能（P2阶段）

| 变现模式 | 描述 | 触发条件 |
|---------|------|---------|
| 广告植入 | 高考/大学相关品牌赞助 | UV > 10万 |
| 付费深度分析 | 更详细的个性化志愿报告 | 免费版用户留存率验证 |
| IP授权 | 毒舌AI人设授权给其他内容 | 人设知名度建立 |

## 8. Go/No-Go 建议

### 8.1 市场验证结论

| 维度 | 结论 |
|------|------|
| 市场需求真实性 | ✅ 大厂集体入场 = 需求真实且巨大 |
| 差异化空间 | ✅ 人格化娱乐内容 = 完全蓝海 |
| B站平台匹配度 | ✅ 年轻用户+吐槽文化 = 天然适配 |
| 时间窗口可行性 | ⚠️ 仅13天，需极简MVP+快速迭代 |
| 风险可控性 | ✅ 四道防线+低分段保护 = 可管理 |

### 8.2 最终建议

> **GO。建议以「引流工具」角色入场，不追求产品独立商业化，聚焦高考窗口期的流量爆发。**
