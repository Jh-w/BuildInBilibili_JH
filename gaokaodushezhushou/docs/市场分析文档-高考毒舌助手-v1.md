## 二、竞争格局分析

### 2.1 大厂产品的共同特征 = 我们的突破口

| 维度 | 所有大厂产品 | 高考毒舌助手 |
|------|------------|------------|
| **产品形态** | 数据查询器 / 匹配计算器 / 报告生成器 | 人格化 AI 对话 + 内容产品 |
| **情感调性** | 中性、客观、官方感 | 毒舌幽默 + 专业分析 + 自嘲式收尾 |
| **用户情绪** | 消除信息焦虑（给更多数据） | 用幽默化解焦虑（让你笑着接受现实） |
| **传播基因** | 几乎为零——没人截图夸克发朋友圈 | 天然传播体——「AI说我625分连门卫微信都加不上」 |
| **人格** | 无性格、无态度 | 有鲜明人设：毒舌但善良，怼分数不怼人 |
| **娱乐性** | 无 | 核心竞争力——不是工具，是内容 |

### 2.2 十篇文章中从未出现的关键词

在 10 篇大厂 AI 高考分析文章中，以下词汇 **零出现**：
- 幽默 / 搞笑 / 有梗
- 人格化 / 人设 / 性格
- 吐槽 / 毒舌 / 怼人
- 娱乐 / 好玩 / 有趣
- 传播 / 病毒 / 出圈
- 情感 / 共情 / 缓解焦虑

> **结论：「AI 高考 × 人格化娱乐内容」是完全的蓝海。没有人做过。**

### 2.3 我们在打什么仗

| 维度 | 大厂战场 | 我们的战场 |
|------|---------|-----------|
| **竞争维度** | 功能覆盖率 → 数据准确性 → 推荐合理性 | 娱乐性 → 传播性 → 人格魅力 |
| **护城河** | 数据量和算法精度 | 人设独特性和内容创作力 |
| **用户关系** | 「我用这个工具查一下分数」 | 「哈哈哈哈这AI太贱了我转给同学」 |
| **商业模式** | 免费工具（为生态引流） | 内容产品（为 B站 账号引流+投币） |

> **核心定位：不跟大厂比谁算得准，跟我们比谁值得截图转发。**

## 三、用户细分

### 3.1 核心用户画像

| 画像 | 描述 | 需求 | 触达方式 |
|------|------|------|---------|
| **🎯 主要用户：高三考生** | 16-19 岁，刚出分，焦虑中，手机重度用户 | 想知道自己能去哪、想被理解、需要情绪出口 | B站 + 评论区 + 同学传播 |
| **🔄 次要用户：大学生** | 19-23 岁，高考过来人，爱看热闹 | 怀旧共鸣、转发给学弟学妹、参与吐槽 | B站推荐流 |
| **👀 围观用户：职场新人** | 23-28 岁，「别人家孩子」的视角 | 社会话题讨论、娱乐消遣 | B站 + 微博话题 |
| **⚠️ 敏感用户：家长群体** | 40-55 岁，考生的父母 | 帮孩子参谋、验证AI推荐靠谱吗 | 微信朋友圈转发（截图卡片） |

### 3.2 用户情绪地图

```
考前 ─── 极度焦虑
考中 ─── 高压释放（6/7-9）
考后到出分前 ─── 等待期的麻木+期待（6/10-22）
出分日 ─── 情绪峰值：狂喜/失落/茫然（6/23）
志愿填报期 ─── 理性+焦虑交织（6/24-7/5）
录取季 ─── 新一轮等待（7/10-8/15）
```

> **我们的产品在情绪最脆弱的那天（出分日）进入。靠幽默化解焦虑，而不是用更多数据加重焦虑。**

## 六、风险与应对

### 6.1 市场风险矩阵

| 风险 | 类型 | 概率 | 影响 | 应对 |
|------|------|------|------|------|
| 大厂快速跟进「人格化」 | 竞争 | 中 | 高 | 人设壁垒——大厂不敢毒舌，品牌调性限制 |
| 舆论反噬（「AI嘲讽考生」） | 舆情 | 中 | 高 | 四道安全防线 + 低分段保护 + 心情选择 |
| 高考后话题枯竭 | 生命周期 | 高 | 中 | 扩展为「大学新生毒舌助手」全周期 |
| 用户信任不足 | 品牌 | 高 | 中 | AI自首机制 + 免责声明 + 数据标注来源 |
| 政策监管收紧 | 合规 | 低 | 高 | 安全红线写死在 system prompt，不做决策替代 |

### 6.2 「毒舌」安全边界

```
✅ 可以做的：
  - 吐槽分数的现实差距（「你这分连门卫微信都加不上」）
  - 幽默翻译专业真相（「生物工程=实验室搬砖四年」）
  - 自嘲式免责（「我是AI不是算命先生」）

❌ 绝对不能做的：
  - 攻击考生本人（「你这种人也配上大学？」）
  - 低分段毒舌（<400分→毒舌全面关闭，鼓励模式）
  - 触碰敏感话题（经济/身体/地域/性别/学校层次羞辱）
```

## 九、结论

> **AI 高考志愿是一个真实且巨大的市场，大厂已经验证。但大厂把功能做满、把灵魂做没——我们的「毒舌」不是在跟它们竞争功能准确度，而是在它们占满的严肃阵地旁边，开了一个它们永远学不会的娱乐阵地。**
>
> **这个产品的最佳角色不是独立参赛者，而是主合集的流量发动机——趁高考 13 天窗口炸流量，全导给主合集。**
>
> **一句话：大厂帮你算出分能去哪，我们让你笑着接受那个答案。**
