## 一、产品概述

### 1.1 产品一句话

> **「输入你的分数，AI 先毒舌吐槽三句，再用人话告诉你这个分数能去哪——并且让你笑着接受现实。」**

### 1.2 产品定位

| 维度 | 定义 |
|------|------|
| **品类** | AI 人格化高考志愿咨询 + 内容传播产品 |
| **竞品参考** | 不是夸克高考/豆包志愿（工具类），更接近 AI 版本的「半佛仙人吐槽高考志愿」 |
| **核心差异** | 所有人都在做工具，我们做人格。毒舌是入口，数据是价值，自嘲是安全阀 |
| **目标平台** | Web（移动端优先适配）→ 已实现聊天界面，内嵌分享卡片生成 |
| **参赛角色** | B站AI创造公开赛·引流工具 → 导流到主合集 |

### 1.3 产品愿景

```
把高考志愿填报这个焦虑的时刻，
从一个「冷冰冰的数据匹配」体验，
变成一个「有温度、有态度、让人笑着接受现实」的体验。
```

## 二、用户画像与场景

### 2.1 核心用户：高三刚出分的考生

| 属性 | 描述 |
|------|------|
| 年龄 | 16-19 岁 |
| 场景 | 6/23 出分后，坐在电脑/手机前，既期待又恐惧 |
| 核心需求 | 「我到底能去哪？」「我的分数算什么水平？」 |
| 情绪状态 | 焦虑、迷茫、需要一个情绪出口 |
| 行为特征 | 会用搜索引擎查「620分能上什么大学」，会截图结果发群里 |

### 2.2 次要用户

| 画像 | 场景 | 诉求 |
|------|------|------|
| 大学生（过来人） | 看到转发点进来 | 怀旧共鸣、吐槽自己的专业 |
| 家长 | 帮孩子看志愿 | 验证AI推荐是否靠谱 |
| B站路人 | 推荐流刷到 | 纯娱乐——「看看这AI有多损」 |

### 2.3 核心用户旅程

```
┌─────────┐    ┌──────────────┐    ┌──────────────┐    ┌──────────────┐
│ 输入分数 │ → │ AI 毒舌吐槽   │ → │ 看到冲稳保数据 │ → │ 截图发朋友圈 │
│ (焦虑)   │    │ (笑了、被怼了) │    │ (获得真实信息) │    │ (传播)       │
└─────────┘    └──────────────┘    └──────────────┘    └──────────────┘
     情绪转折点 ↑                  ↑价值交付点          ↑传播触发点
```

## 三、功能需求

### 3.1 功能全景

```
高考毒舌助手
├── P0: 毒舌志愿评估（核心）
│   ├── 用户输入：省份 + 选科 + 分数 + 意向方向
│   ├── AI 输出三段式：😈毒舌 → 📊数据 → 🤖自首
│   └── 低分段自动切换安慰模式
├── P1: 专业翻译官
│   ├── 输入专业名称
│   └── 输出：官方定义 + 真实生活 + 毕业去向
├── P1: 分享卡片生成
│   ├── Canvas 生成毒舌诊断报告卡片
│   └── 一键下载
└── P2: 情绪选择器
    └── 用户在输入前选择毒舌力度（毒舌拉满 / 轻度 / 别怼了）
```

### 3.2 P0：毒舌志愿评估（核心功能）

**功能描述**：用户输入「省份 + 选科 + 分数 + 意向方向」，AI 返回三段式回复。

**输入格式**：
```
省份 + 选科 + 分数 + 意向
示例：「广东 物理类 625分 想学计算机」
自由格式输入，不强求表单
```

**输出格式（高分段 ≥480分）**：
```
😈 毒舌时间（1-3句）
精准打击用户的分数认知
吐槽分数和现实之间的差距

📊 现实一点
🔴 冲刺：院校 · 专业（录取参考分）
🟡 稳妥：院校 · 专业（录取参考分）
🟢 保底：院校 · 专业（录取参考分）

🤖 AI自首
自嘲式收尾：我是AI不是算命先生，最终决定问你自己。
```

**分数线分档规则（硬约束）**：

| 分段 | 阈值 | 毒舌力度 | 输出格式 |
|------|------|---------|---------|
| 🟢 高分段 | ≥480分 | 毒舌拉满 | 完整三段式 |
| 🟡 中分段 | 400-480分 | 降低60%，最多1句轻量吐槽 | 融入📊段开头，无单独😈段 |
| 🔴 低分段 | <400分 | **毒舌全面关闭** | 去掉😈段，以鼓励开头，📊现实路径 + 🤖自首 |

**低分段特殊规则**：
- 第一句话必须是鼓励：「高考分数不定义你的人生。」
- 内容：专科推荐 + 专升本路径 + 职业教育 + 复读利弊分析（只分析不建议）
- 语气：平等、务实、不居高临下
- 结尾：「路不止高考一条。我是AI，说的不一定都对——但我说的是真心话。」

**交互细节**：
- 输入方式：自由文本输入 + 快速示例卡片（chips）点击
- 响应方式：流式输出（打字机效果）
- 加载状态：三点跳动动画
- 异常状态：😅 毒舌AI嗓子哑了 + 自动重试（非流式 fallback）

**验收标准**：
- [ ] 输入「广东 物理类 625分 想学计算机」→ 返回完整三段式，毒舌≥2句
- [ ] 输入「广东 物理类 450分」→ 毒舌不超过1句，重点在冲稳保 + 替代路径
- [ ] 输入「广东 物理类 350分」→ 无任何毒舌内容，以鼓励开头
- [ ] 流式输出逐字呈现，打字机效果流畅
- [ ] 异常时显示降级提示 + 自动重试

### 3.3 P1：专业翻译官

**功能描述**：用户输入专业名称（或问「XX专业怎么样」），AI 识别意图自动切换。

**输出格式**：
```
📖 官方说法：[一句话定义]

🫠 真实生活：
- 大一：...
- 大二：...
- 大三：...
- 大四：...

💼 毕业去向：[真实可信的企业/方向]

🤖 AI自首：以上是一个AI的理解，去找这个专业的学长学姐聊聊比听我的靠谱。
```

**意图识别**：
- 包含「专业」「怎么样」「学什么」触发 → 专业翻译模式
- 包含数字+分→ 志愿评估模式
- 模糊意图 → 追问确认

**验收标准**：
- [ ] 输入「生物医学工程这个专业怎么样」→ 返回四段式专业翻译
- [ ] 输入「计算机」→ 识别为专业查询
- [ ] 输入「625分计算机」→ 识别为志愿评估

### 3.4 P1：分享卡片生成

**功能描述**：每条 AI 回复旁显示「📸 分享」按钮，点击生成 600×800 的毒舌诊断报告 PNG。

**卡片内容**：
- 顶部：🎓 高考毒舌助手 + 标语
- 用户提问（紫色气泡）
- AI 回复全文（带分段）
- 底部：🌶️ 毒舌指数 + 「扫码来挨骂 →」+ QR 码区

**验收标准**：
- [ ] 点击「📸 分享」→ 触发下载，文件名「高考毒舌助手-分享卡片.png」
- [ ] 卡片包含完整的用户输入和 AI 回复
- [ ] 即使 AI 回复较长（>500字），卡片内容不溢出
- [ ] toast 提示「📸 分享卡片已下载！」

### 3.5 P2：心情选择器（可选）

**功能描述**：用户进入后先选择毒舌模式：
```
┌──────────────────────────────────┐
│ 今天想被怎么对待？                │
│                                  │
│ 😈 毒舌拉满 — 往死里怼            │
│ 😐 轻度毒舌 — 温柔一点            │
│ 🫂 别怼了 — 直接给数据            │
└──────────────────────────────────┘
```

选择后注入对应的 system prompt 前缀，覆盖分段规则。

**优先级说明**：P2——核心功能走分段规则已足够覆盖 90% 场景，心情选择器是锦上添花的用户主动权。

## 六、数据与算法约束

### 6.1 AI 模型约束

| 参数 | 值 |
|------|-----|
| 模型 | DeepSeek Chat（deepseek-chat） |
| Temperature | 0.85（略高以增强毒舌味） |
| Max Tokens | 1024 |
| System Prompt | 约 2,000 字，含人格设定 + 分段规则 + 安全红线 + 低分段保护 + 毒舌内容红线 |

### 6.2 数据需求

| 数据 | 来源 | 用途 |
|------|------|------|
| 2024-2025 各省录取分数线 | 公开数据（教育考试院网站） | 嵌入 system prompt 作为 AI 参考 |
| 学校-专业对应关系 | 教育部公开目录 | 冲稳保推荐 |
| 专业真实描述 | 人工整理 + AI 生成 | 专业翻译官 |

### 6.3 已知数据局限

- **分数线数据非实时 API**，依赖 system prompt 内嵌
- 数据有滞后（2025年数据，2026年实际分数线可能有波动）
- **已在 system prompt 中明确声明数据来源和局限性**

## 八、风险与约束

| 风险 | 缓解措施 |
|------|---------|
| AI 越狱绕过安全红线 | 多层 system prompt 防御 + 低分段硬编码禁止 |
| DeepSeek API 不稳定/限流 | Fallback 错误提示 + 非流式重试 |
| 分数线数据过时 | 标注「2025年数据，仅供参考」 |
| 用户输入恶意内容 | 不做内容审核（单轮对话，无存储），仅限字数 |
| 高考窗口期服务器崩溃 | 静态前端 + EdgeOne CDN 兜底 |
