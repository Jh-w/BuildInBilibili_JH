# 🧬 A 选项：AI 身体翻译官 — 完整产品方案

> 定位：BME科班出品 · 用AI把你身体的每个器官拟人化，让它们能跟你聊天
> 参赛角色：🏆 主合集首选

---

## 一、产品一句话

**「给你的每个器官开一个聊天窗口——心脏骂你熬夜，胃吐槽你吃辣，膝盖求你少跑点。」**

## 二、为什么这个产品只有你能做

| 别人做 | 你做 |
|--------|------|
| 器官拟人化靠脑补和百度百科 | 器官拟人化有解剖学、生理学、病理学依据 |
| 「心脏很累」← 这句谁都会写 | 「你昨晚深睡只有42分钟，我今天泵血泵得像跑了半个马」← 这句需要BME |
| 人格设计靠直觉 | 人格设计有学术底座——交感神经/副交感神经、血流动力学、代谢通路 |

**核心壁垒**：别人能复制「AI + 器官拟人化」的概念，但复制不了 BME 7 年的内容深度。

## 三、核心功能

### 3.1 器官角色体系（MVP 6个器官）

| 器官 | 人格 | 口头禅 | 科学依据 |
|------|------|--------|---------|
| **心脏 ❤️** | 暴躁打工人 | 「你熬夜我加班，你有良心吗」 | 交感神经兴奋 → 心率↑ → 心脏负荷 |
| **胃 🫗** | 矫情美食家 | 「又吃辣？你是不是想换一个胃」 | 辣椒素 → TRPV1受体 → 胃黏膜刺激 |
| **肺 🫁** | 冷静观察者 | 「你跑步喘成狗，怪我咯？」 | 最大摄氧量 VO₂max、通气量 |
| **膝盖 🦵** | 退休老兵 | 「再跑10公里我就申请残疾退休」 | 关节软骨磨损、半月板压力分布 |
| **大脑 🧠** | 阴阳怪气CEO | 「你让我凌晨三点还在刷手机，明天别怪效率低」 | 蓝光→褪黑素抑制→睡眠质量 |
| **肝 🍺** | 沉默社畜 | 「我不说话不代表我没在加班解毒」 | 酒精代谢→乙醛→肝细胞损伤 |

### 3.2 用户交互流程

```
用户打开产品
  → 看到6个器官的聊天列表（像微信）
  → 点击任一器官 → 进入对话
  → 器官根据「用户画像」+「当前时段」主动发起对话
  → 用户可以打字/语音跟器官聊天
  → 器官回复带「科学解释」小标签（点击展开）
  → 分享卡片：「我的心脏今天说了：_____」
```

### 3.3 三个核心交互模式

**模式1：主动问候（定时触发）**
- 早上8点：心脏：「醒了？昨晚你睡了5小时，我今天继续过劳肥。」
- 中午12点：胃：「这个点该吃饭了，别给我塞麻辣烫。」
- 晚上11点：大脑：「又刷手机？明天的你会恨今晚的你。」

**模式2：事件吐槽（用户输入触发）**
- 用户：「今天跑了10公里」→ 膝盖：「10公里？你知道你78公斤的体重对我来说意味着什么吗？」
- 用户：「喝了两瓶啤酒」→ 肝：「（沉默5秒）……行吧，今晚加班。」

**模式3：器官互怼（多器官联动）**
- 胃：「这顿火锅我很满意」→ 肝：「你满意个屁，你知道辣椒+酒精我要加班多久吗？」

## 四、技术架构

| 层 | 技术 | 说明 |
|----|------|------|
| 前端 | React + TypeScript + Vite | 复刻微信聊天界面 |
| 样式 | 聊天列表 + 聊天气泡 + 器官头像（emoji + 配色） | 零设计门槛，像素级复刻微信 |
| AI | DeepSeek API（system prompt = 器官人设 + 科学约束） | 每个器官一个 system prompt |
| 知识库 | BME专业知识的 RAG 检索（可选，MVP可省略） | 确保科学准确性 |
| 部署 | 腾讯云 EdgeOne Pages + SCF | 跟跑步人格测试同架构 |
| 分享 | Canvas 生成分享卡片 | 复刻跑步人格测试的分享链路 |

### System Prompt 示例（心脏）

```
你是一颗人类心脏，你的人格设定：
- 暴躁、务实、嘴硬心软
- 你非常了解心血管生理学
- 你说话带数据：心率、血压、睡眠时长
- 你会吐槽主人的生活习惯，但最终关心TA的健康
- 回复控制在80字以内，带一句科学事实
- 格式：回复正文 + （📚 科学解释标签：一句话解释背后的生理学原理）

禁止：
- 不要给出医疗建议（吃药、去医院等）
- 不要说教超过两句
- 不要假装你是医生
```

## 七、风险

| 风险 | 应对 |
|------|------|
| 器官「说错话」造成科学错误 | system prompt 里硬约束科学准确性，加一个「事实核查」校验层 |
| 用户把器官聊天当医疗建议 | 每句回复底部加固定免责：「我是一个AI拟人，不是医生。请咨询专业医疗人员」 |
| 开发14天，比赛开始后2周才上线 | EP0提前发（6/11），开发期内就有内容输出 |
